\section{Тестирование}

\subsection{Аппаратно-программная конфигурация}

Тестирование проводилось на двух ноутбуках:
\begin{itemize}
  \item Acer Aspire 5 с процессором Intel Core i3 1215U и видеокартой Intel UHD Graphics;
  \item HP Pavilion 14 с ОС Ubuntu 22. В ноутбуке установлены Intel Core i7 1355U и Intel Iris Xe Graphics.
\end{itemize}

\subsection{Методика тестирования}

Для тестирования минимального учебного компьютера необходимо проверить работу арифметической операции, перехода по условию и функцию переноса данных между ячейками памяти.

Задачи для тестирования минимальной ОС:
\begin{itemize}
  \item выполнение команд во время параллельного расчета чисел Фибоначчи;
  \item создание, изменение, перемещение и удаление текстового файла;
  \item прием числа с клавиатуры и вывод его на экран;
  \item компиляция кода на C и исполнение полученной программы.
\end{itemize}

Для тестирования проекта эмулятора процессора RISC-V были поставлены следующие задачи:

\begin{itemize}
  \item Проверка корректности выполнения арифметических операций (сложение, вычитание, логические операции, сдвиги).
  \item Тестирование переходов по условию и безусловных переходов с использованием меток.
  \item Проверка работы с памятью: загрузка и сохранение данных различных размеров (байт, слова).
  \item Тестирование системных вызовов через инструкцию \texttt{ECALL}, включая ввод-вывод и завершение работы.
  \item Проверка работы псевдоинструкций и правильности их трансляции в базовые инструкции.
  \item Запуск простых программ, например, вычисление чисел Фибоначчи, бесконечные циклы, ввод и вывод чисел.
  \item Проверка обработки ошибок и корректного завершения эмуляции.
\end{itemize}

Для комплексного тестирования использовался следующий подход:

\begin{enumerate}
  \item Сборка тестовых программ в ассемблере с помощью Python-скрипта, переводящего их в hex-код.
  \item Конвертация hex-файлов в бинарный формат для загрузки в эмулятор.
  \item Запуск программ в эмуляторе и сравнение результатов с ожидаемыми.
  \item Проверка корректности работы регистров, памяти и системы прерываний.
  \item Отслеживание выполнения инструкций и отладка на уровне машинного кода.
\end{enumerate}

\section{Заключение}

  Для достижения цели выпускной квалификационной работы были получены следующие
  результаты
  \begin{itemize}
    \item Выполнен сравнительный обзор существующих образовательных комплексов, оценка их документации и выявление их практической ценности
    \item Разработаны виртуальные процессоры с возможностью моделирования и тестирования
    \item Запущена минимальная операционная система (упрощённый xv6) на виртуальном устройстве
    \item Подготовлены документация и учебные материалы для преподавателей и студентов
    \item Проведено тестирование
  \end{itemize}
